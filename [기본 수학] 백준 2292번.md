#### 코드
```py
'''
벌집
1층 : 1
2층 : 2~7     #6개
3층 : 8~19    #12개
4층 : 20~37   # 18개 : 6개씩 증가

1+6
1+6+6+6
1+6+6+6+6+6+6
1 3 6 10 15 21

1층 = 1번
2층 = 2번
3층 = 3번
4층 = 네번 방을 지나감  # 층수 = 방을 지나간 수
'''


n = int(input())
x = 1
i = 0

while 1:
    x += 6*i
    i +=1

    if(x>=n):
        print(i)
        break
    elif(n==1):
        print(1)
        break


```

#### Point
n번 방으로 가는 최소 값은 그 방이 해당하는 층수이다.
x = 각 층수의 최댓값을 계산한 값으로 x값이 n값을 넘게 되거나 같아지면 i값(층수)을 출력한다
